# Customer Data Analysis Project

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Olist customer dataset using Python. The main objective of this project is to understand customer distribution patterns across different states and cities and generate meaningful business insights from raw data.

The dataset was cleaned, analyzed, and visualized using various Python libraries such as Pandas, Matplotlib, and Seaborn.

---

# 🎯 Objective

* Understand customer distribution across states and cities
* Perform data cleaning and preprocessing
* Identify missing values and duplicate records
* Create visualizations to analyze customer patterns
* Generate business insights from customer data

---

* Jupyter Notebook

* Seaborn

# 🔍 Operations Performed

## ✅ Data Loading

* Loaded dataset using Pandas
* Verified dataset files using Python

## ✅ Data Understanding

* Viewed first 5 rows using `head()`
* Checked dataset information using `info()`
* Displayed column names using `columns`

## ✅ Data Cleaning

* Checked missing values using `isnull().sum()`
* Identified duplicate rows using `duplicated().sum()`
* Removed duplicate records using `drop_duplicates()`

## ✅ Data Analysis

* Analyzed customer distribution by state
* Analyzed customer distribution by city
* Counted unique states and cities
* Calculated customer frequency across regions

## ✅ Data Visualization

Created multiple visualizations including:

* Bar Charts
* Horizontal Bar Charts
* Pie Charts

# ✅ Conclusion

The Exploratory Data Analysis performed on the Olist customer dataset helped identify customer distribution patterns across different states and cities. The dataset was cleaned by checking missing values and removing duplicate records to improve data quality.

The analysis revealed that customer concentration is higher in a few major states and cities, indicating stronger market presence in those regions. Various visualizations such as bar charts and pie charts were used to better understand customer distribution and regional trends.

This project demonstrated important data analytics skills including data cleaning, data exploration, visualization, and extracting meaningful business insights using Python libraries like Pandas, Matplotlib, and Seaborn.

Overall, the project highlights how raw customer data can be transformed into useful information for business understanding and decision-making.
